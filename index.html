Tutorial 1 - Block, Inline and Breaks

In HTML Elements are containers for information, typically text or images. Each Element can be categorized as a Block Element or an In-line Element.

  • Block: These elements create a new vertical block, stacking content ontop of each other
  • In-line: These elements create a new horiztonal block, keeping content in a horizontal line with each other.

There are many different Elements in HTML, but with clever use of Block and In-line elements, you can easily structure the layout of your content.

Additionally, you can use the br element inbetween in-line elements to Break them up, causing them to stack like Block elements.
